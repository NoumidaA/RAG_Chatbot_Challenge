# PDF RAG Chatbot

Open-source RAG chatbot for large PDF corpora.

## Features
- Native PDF text extraction
- OCR for scanned pages
- Chunking with metadata
- FAISS vector search
- Hugging Face generation model
- Source citations by file and page
